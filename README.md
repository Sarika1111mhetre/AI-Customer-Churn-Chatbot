# 🤖 AI Customer Churn Chatbot

## 📌 Overview

An AI-powered chatbot that allows users to analyze customer churn data using natural language queries.

The system converts user questions into SQL, retrieves data from PostgreSQL, and displays insights through interactive charts and tables.

## 🚀 Features

* Natural Language to SQL conversion
* Interactive Streamlit dashboard
* PostgreSQL integration
* Data visualization & insights

## 🛠️ Tech Stack

Python • Streamlit • PostgreSQL • Pandas • Matplotlib

## ⚙️ Setup

```bash
git clone https://github.com/your-username/ai-customer-churn-chatbot.git
cd ai-customer-churn-chatbot
pip install -r requirements.txt
streamlit run app.py
```

## 🔐 Environment Variables

Create a `.env` file:

DB_USER=your_username
DB_PASSWORD=your_password
DB_HOST=localhost
DB_NAME=ai_churn_chatbot
